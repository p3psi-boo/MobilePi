Phase 1 全部实现：

1. **_OutputViewState 状态字段**：_expandedTurnKeys (Set<int>)、_collapseVersion (int)、_shouldStickToBottom (bool)、_programmaticScroll (bool)、_autoScrollThreshold (50px)

2. **Turn Collapsing**：_OutputViewState.build() 内的 Selector builder 中预计算 lastNonUserIdx，除最新 turn 和手动展开的 turn 外，所有 _AgentTurnWidget 折叠为一行的 _CollapsedTurnRow（显示：agent avatar + text preview + tool count + expand 箭头）

3. **_CollapsedTurnRow**：新 StatelessWidget，紧凑单行预览，tap 触发 onExpand 回调展开 turn 并停止 stick-to-bottom 滚动

4. **Fold All 按钮**：Stack 底部的小图标按钮，仅在有手动展开 turn 时显示，点击清除 _expandedTurnKeys + 回到 bottom

5. **Scroll-to-bottom 按钮**：Stack 底部的小图标按钮，当用户向上滚动离开底部时显示

6. **Stick-to-bottom 状态机**：
   - _onScroll：更新 _shouldStickToBottom 为 (max-pixels < threshold)，有变化时 setState
   - _scrollToBottom：设置 _programmaticScroll=true 防干扰，jumpTo 后设回 false
   - _onProviderChanged：仅 _shouldStickToBottom==true 时滚动
